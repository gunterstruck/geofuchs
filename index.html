Datenschutzrichtlinie – SeniorSafe

Die App SeniorSafe dient ausschließlich der lokalen Notfallhilfe für Seniorinnen und Senioren.

1. Keine Datenübertragung an Dritte:
Es werden keine personenbezogenen Daten an externe Server übermittelt. Alle sensiblen Informationen bleiben auf dem Gerät.

2. Berechtigungen & Zweck:

  • Standortdaten: Nur zur Notfallübermittlung an vom Nutzer hinterlegte Kontakte.
  • SMS-Versand: Wird ausschließlich verwendet, um im Notfall die Vertrauensperson zu benachrichtigen – nach expliziter Zustimmung.
  • Push-Benachrichtigungen: Nur für systemkritische Hinweise (keine Werbung).
3. Keine Profilbildung, kein Tracking, keine Werbung.
4. Ihre Rechte: Sie können jederzeit Auskunft, Berichtigung oder Löschung Ihrer lokal gespeicherten Daten verlangen. Alle Funktionen sind DSGVO-konform umgesetzt.

Weitere Informationen zum Datenschutz erhalten Sie per E-Mail-Anfrage (siehe Impressum).

Datenschutzrichtlinie – GeoFuchs

Die App GeoFuchs verwendet ausschließlich öffentliche Kartendaten (OpenStreetMap, GeoJSON) zur Darstellung von Geoinformationen.

1. Keine Erhebung personenbezogener Daten:
Die Nutzung ist ohne Login, ohne Erhebung von Namen, E-Mail-Adressen oder Standortdaten möglich.

2. Keine Tracking-Tools oder Analyse-Dienste:
GeoFuchs integriert keinerlei Google Analytics, Facebook Pixel oder ähnliche Mechanismen.

3. Externe Links & Kartenanbieter:
Beim Anzeigen der Karte wird eine Verbindung zu OpenStreetMap hergestellt. Es gelten die Datenschutzrichtlinien von OpenStreetMap.

4. Ihre Rechte: Sie haben das Recht auf Auskunft, Berichtigung oder Löschung aller eventuell erfassten Daten (was jedoch standardmäßig nicht erfolgt). Für Nachfragen siehe Impressum.
